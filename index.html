若要在hexo中引用图片，需要在source目录下面新建一个目录images，然后把图片放到目录中，然后在markdown中直接![](/images/picture_name)就行了。

我常用的Chrome 插件

发表于 2015-12-29   |   分类于 效率

说明

简单记录一下自己在Chrome浏览器下常用的插件，会经常更新的。

我所用的Chrome如下图所示，

  • Google Translate
    这个插件可以在网页中进行实时的翻译，因为我用的linux系统，感觉没有一个特别满意的翻译软件，所以，暂时用的这个。
  • Auto Copy
    这个插件顾名思义，就是帮我省了一步操作，在选中一段话的时候，它会自己复制，不用去 右键 -> copy 了，这个插件再配上鼠标上的中键，感觉复制粘贴起来方便多了。
  • 网易云音乐
    这个插件在Chrome的商店里下不到，是github上一个哥们做的，确实很良心，用这个插件在linux下用网页版网易云听音乐可以实现全局的快捷键。
  • OneTab
    我一般是在前一天要处理的问题没解决，打开了好多网页第二天不想重新找的时候用这个，这个可以可以把所以看过的网页都保存起来。但是好像官方给出的使用方法不是这样的，算了，不管了。
  • Vimium
    键盘流必备，基本可以实现用键盘来浏览网页。
  • Infinity New Tab

    这个插件修改了Chrome的起始页，做的真的超级好啊，把各种常用的网站加入到起始页，方便的很，而且各种动画效果做的也很流畅。
  • Google Docs Offine
    这个玩意也不是太了解是干啥的，好像是在用Google文档在线编辑的时候，必须有这个插件，才能复制粘贴？不是太了解啦。
  • Bookmark Manager
    这个是谷歌官方出的新收藏夹，挺赞的。
  • SmoothScroll
    Chrome浏览器在滚动的时候是在是太难受了，一蹦一蹦的。这个插件可以使Chrome浏览器在滚动的时候像苹果那样流畅，感觉舒服多了。
  • Black Black Chrome Theme Dark
    一款基本全黑的Chrome主题。
  • Thin Scroll Bar
    这个插件可以让Chrome浏览器的滚动条变的很窄，不占空间了。
  • Evernote Web Clipper
    这个插件是印象笔记官方出的，可以一键把当前正在看的页面保存到印象笔记中。
  • 替换字体的中文部分为雅黑
    这个插件是在用Windows系统的时候使用的，Chrome在浏览网页时，好多网页的字体都是宋体，感觉实在是太难看了。微软雅黑字体还是不错了，正好发现这个插件，就果断安装了。
  • 保护眼睛
    好像是调整背景色，好久没用过了。
  • Clearly
    这个就是把当前的页面给处理一下，把没用的去掉，方便阅读。
  • Any.do Extension
    还真忘了是干啥的了。
  • Context Menu Search
    一般来说有，右键只有用默认的引擎进行搜索，这个插件可以给右键搜索添加多个搜索引擎，在查找资料的时候很用的上。
  • Proxy SwitchOmega
    用来修改Chrome浏览器的代理，非常方便。

”修改键盘映射“

发表于 2015-12-29   |   分类于 效率

说明

感觉也是出于个人需求吧，现在市场上的键盘布局感觉不适合经常编程的人使用，所以想办法给修改一下。

要修改的键有 ESC，CapsLock，control_L，control_R，Alt_R（后面的L和R表示的是左边还是右边）
修改后的功能映射为：

  • CapsLock 用作 Control_L
  • Control_L 用作 ESC
  • Alt_R 用作 Control_R
  • Control_R 用作 CapsLock

操作

  1. 随便找个目录，新建一个文件，输入以下代码：
!以下代码是改变了一下键盘的映射
!把
! CapsLock映射为Control_L
! Control_L映射为Escape
! Alt_R映射为Control_R
! Control_R映射为CapsLock
!
remove Lock = Caps_Lock
remove Control = Control_L
remove Control = Control_R
remove mod1 = Alt_R
keysym Caps_Lock = Control_L
keysym Control_L = Escape
keysym Alt_R = Control_R
keysym Control_R = Caps_Lock
add Lock = Caps_Lock
add Control = Control_L
add mod1 = Alt_R
add Control = Control_R

  1. 执行 xmodmap YOUR_FILENAME，就行了。

  2. 如果要想开机执行这段脚本，只需要把文件名改为“.xmodmaprc”，然后放在~ 名录下即可
